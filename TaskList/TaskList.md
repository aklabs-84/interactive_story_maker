# 개발 태스크 리스트 (TaskList)

이 리스트는 인터랙티브 스토리 메이커 프로젝트의 유지보수 및 추가 기능 개발을 위한 단계별 작업 목록입니다.

## 1단계: 기본 인프라 보강
- [ ] `config.js` 설정 가이드 문서화 (GAS URL 설정 방법 등)
- [ ] 정적 자원(이미지, 사운드) 에셋 최적화 및 기본 샘플 라이브러리 구축
- [ ] 다양한 브라우저에서의 LocalStorage 호환성 테스트

## 2단계: 에디터 기능 고도화
- [ ] 드래그 앤 드롭 기반의 스토리 맵 뷰어(Visualization) 통합
- [ ] 실시간 글자수 제한 및 유효성 검사 기능 강화
- [ ] 실행 취소/다시 실행 (Undo/Redo) 기능 구현
- [ ] 자동 저장 (Auto-save) 기능 추가

## 3단계: 플레이어 경험 향상
- [ ] 장면 전환 애니메이션 다양화 (페이드, 슬라이드 등)
- [ ] 중간 저장 기능 (Save Point/Checkpoint) 구현
- [ ] 텍스트 타이핑 효과(Typewriter effect) 옵션 추가

## 4단계: 관리 및 공유 최적화
- [ ] 스토리 목록 검색 및 필터링 기능
- [ ] Google Sheets 동기화 오류 시 복구 로직 강화
- [ ] QR 코드 기반 스토리 공유 기능 추가

## 5단계: 배포 및 운영
- [ ] GitHub Pages를 이용한 정적 호스팅 설정
- [ ] 교육용 튜토리얼 스토리 제작 및 기본 배포
- [ ] 사용자 피드백 수집 기능 (설문 조사 링크 등)
