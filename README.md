# 인터랙티브 스토리 메이커

선택에 따라 이야기가 달라지는 인터랙티브 스토리를 만들고, 체험하고, 공유할 수 있는 웹앱입니다. 에디터에서 스토리를 설계하고, 플레이어로 바로 실행해보며, 저장소에서 스토리를 관리할 수 있습니다.

## 주요 기능
- 스토리 만들기: 에디터에서 선택지 기반 스토리를 설계하고 저장
- 스토리 체험: 만든 스토리를 플레이어에서 즉시 실행
- 내 스토리: 저장된 스토리 관리·공유
- 스토리 설계 GEM: 외부 GEM 도구로 아이디어 구체화
- 스토리 맵: 스토리 흐름을 한눈에 설계할 수 있는 맵 페이지로 이동 ([바로가기](https://storywaver.vercel.app/))
- 라이트/다크 모드: 모드 토글로 선호하는 테마 적용

## 빠르게 시작하기
1) 이 저장소를 클론하거나 ZIP으로 다운로드합니다.
2) `index.html`을 브라우저로 열면 메인 페이지가 표시됩니다.
3) 원하는 카드(스토리 만들기/체험/내 스토리/스토리 설계 GEM/스토리 맵)를 선택해 바로 이동합니다.

## 폴더 구성
- `index.html`: 메인 랜딩 페이지
- `editor.html`: 스토리 편집기
- `player.html`: 스토리 플레이어
- `manager.html`: 스토리 관리 페이지
- `css/common.css`: 테마 및 공통 스타일
- `js/`: 설정, 테마, 유틸 스크립트

## 호환성
모던 브라우저(Chrome, Edge, Safari, Firefox) 최신 버전을 권장합니다.
